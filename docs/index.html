Características

Todo lo que necesitás

Análisis Bidireccional

Encuentra CSS sin blockClass y blockClass sin CSS. Doble chequeo para no dejar nada suelto.

Limpieza Interactiva

El comando fix te guía regla por regla. Vos decidís qué se elimina y qué se conserva.

Informes Detallados

Genera reportes en Markdown con cada análisis y sesión de limpieza para un registro histórico.

Seguro

Ignora automáticamente los archivos CSS de componentes React custom para evitar falsos positivos.

Inteligente

Reconoce clases de estado dinámicas de VTEX como --isActive y solo valida el blockClass principal.

CLI + GUI

Usalo desde la terminal o con la aplicación de escritorio. Elegí la que mejor se adapte a tu workflow.
